# TimeClient

## API

- TimeClient()
- void Init()
- uint8_t GetHours()
- uint8_t GetMinutes()

***

**TimeClient()**

Creates UDP and NTP client objects.

***

**void Init()**

Initializes the client. Sets the time zone. Sets the frequency of time updates.

***

**uint8_t GetHours()**

Returns the hours.

***

**uint8_t GetMinutes()**

Returns the minutes.
